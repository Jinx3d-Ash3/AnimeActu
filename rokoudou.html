"Rokoudou No Onna-Tacchi" nouveau visuel


"Rokudō no Onna-tachi" ou Les méchantes filles de Rokudō est une série de manga japonaise écrite et illustrée par Yūji Nakamura. Le manga a été sérialisé dans le Weekly Shōnen Champion d'Akita Shoten de juin 2016 à avril 2021, et ses chapitres ont été rassemblés en vingt-six volumes tankōbon. Une adaptation en série télévisée animée par Satelight est prévue pour avril 2023. C'est Keiya Saito qui réalise le projet avec Yuichiro Momose ( So I’m a Spider, So What? ) signe la composition scénaristique tandis que Shinya Segawa ( Dark Gathering ) est crédité en tant que character designer. Hiroyuki Taiga( Sakugan ) propose le design des monstres.

Voici la bande d'annonce :

Synopsys

Le titre suit le quotidien mouvementé de Tosuke Rokudo, un lycéen membre d’une école contrôlée par des racailles. Harcelé en permanence, la vie du jeune homme change du tout au tout lorsqu’il obtient une ancienne relique magique qui le rend irrésistible aux yeux des femmes ! Avec toutes les délinquantes du lycée de son côté, dont Ranna Himawari, la plus terrible de toutes, plus rien n’y personne ne peut l’inquiéter !


L'opening est ENDLESS LABYRYNTH interprete par le groupe japonais EverdreaM et l'ending est du groupe THE RAMPAGE qui interprete la chanson Love will find a way.

D'apres le trailer, Le cast est composée pou l'instant de:

• Gen Sato, dans le rôle de Tosuke Rokudo.

• Sumire Uesaka, dans le rôle de Ranna Himawari.

• Naomi Ozora, dans le rôle de Sayuri Osanada.

• Yuki Yomichi, dans le rôle de Azami Himeno.

• Misato Matsuoka, dans le rôle de Tsubaki.

• Hitomi Sekine, dans le rôle de Mizue Tsuyukusa.

• Haruki Ishiya, dans le rôle de Haruya Iinuma.

• Daisuke Namikawa, dans le rôle de Masaru Hinomoto.

• Akira Ishida, dans le rôle de Kota Kijima.
